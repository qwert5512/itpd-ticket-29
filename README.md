---

## Опис завдання

### Задача 1
Іванчик побачив у торговому автоматі свій улюблений напій.  
Відомо:
- `a` — ціна однієї пляшки напою (грн);
- `b` — кількість грошей, яку має Іванчик;
- `c` — кількість пляшок напою в автоматі.

Необхідно визначити максимальну кількість пляшок, яку Іванчик може купити,  
враховуючи наявні кошти та обмежену кількість товару.

Розв’язання реалізовано мовою Python з використанням функцій.

---

## Алгоритм розв’язання

1. Ввести значення `a`, `b`, `c`
2. Обчислити кількість пляшок, які можна купити за гроші (`b // a`)
3. Порівняти отримане значення з кількістю пляшок в автоматі
4. Вивести мінімальне з двох значень

---

## Приклад роботи програми

Вхідні дані:

a = 10

b = 35

c = 100

Вихідні дані:

Максимальна кількість пляшок: 3
