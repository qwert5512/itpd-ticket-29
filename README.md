Білет №29

Інформаційні технології в професійній діяльності

    Цей репозиторій містить розв’язання практичного завдання з екзаменаційного білета №29
    
з дисципліни «Інформаційні технології в професійній діяльності».

    Іванчик побачив у торговому автоматі свій улюблений напій. Відомо, що a — ціна однієї
    
пляшки напою (грн), b — кількість грошей, яку має Іванчик, c — кількість пляшок напою
в автоматі. Необхідно визначити максимальну кількість пляшок, яку Іванчик може купити,
враховуючи наявні кошти та обмежену кількість товару.

    Розв’язання задачі реалізовано мовою Python з використанням функцій.

    Алгоритм розв’язання полягає у введенні значень a, b та c, обчисленні кількості
    
пляшок, які можна купити за гроші шляхом цілочисельного ділення b // a, порівнянні
отриманого значення з кількістю пляшок в автоматі та визначенні мінімального з двох
значень.

    Приклад роботи програми.

Вхідні дані:

a = 10

b = 35

c = 100


Вихідні дані:

Максимальна кількість пляшок: 3
